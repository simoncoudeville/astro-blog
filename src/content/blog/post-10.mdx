For quite some time now I use this code snippet to set the default or global font size for the entire design of my page:

```css
:root {
  --root-font-size: 20;
  font-size: calc(var(--root-font-size) / 16 * 100%);
}
```

In this case it makes sure that every element in the html that doesn't have a specific font-size set, like a standard piece e.g. a `<p>` or `<ul>`, has a font-size of 125%. Which results in 20px with standard browser settings. Because 125% of 16px is 20px. It's a designer's opinionated choice on how the page should look. This also determines what 1rem equals to. I then use this value to calculate everything else, from font sizes to whitespace. It just makes sense to me.

Maar ik heb altijd geweten dat deze berekening één grote fout heeft. Ik gebruik een vaste waarde van 16 om de font size te berekenen. Omdat dit de standaard lettergrootte is voor webbrowsers. Maar dit houdt geen rekening met de gebruikersvoorkeuren voor de lettergrootte. Als een gebruiker z'n browser font size instelt op bv medium in Chrome, zou de globale font-size 20px moeten zijn. Dat is zo als je als author niet zelf een font size beslist voor je site. Maar als je dat wel doet zoals ik doe wordt dat de globale font-size niet 20px maar 125% van 20px = 25px is. En al de rest die gebaseerd is op 1rem ook opgeblazen wordt. Dat is niet wat ik wil. Als de gebruiker wil dat de font-size 20px is en de designer wil dat ook, waarom kan het resultaat dan niet 20px zijn?

Tijdens mijn recente onderzoek over fluid typography belandde ik op de website van Miriam Suzanne en de verschillende artikels die zij er over geschreven heeft. En net zoals zij is ze op zoek naar een manier om user preferences en author design choices op elkaar af te stemmen. Dat zette me aan het denken. Wat we zouden moeten kunnen doen is kijken naar de font size die de gebruiker heeft ingesteld in zijn browser en die gebruiken als basis voor de berekening in plaats van de vaste 16px die ik nu gebruik. In eerste instantie dacht ik dat als de 16 zou vervangen worden door de 


```css
:root {
  --root-font-size: 20;
  font-size: calc(var(--root-font-size) / var(--user-font-size) * 100%);
}
```
